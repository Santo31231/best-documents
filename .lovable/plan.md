## Сайт услуг по легализации документов

Современный одностраничный сайт в стиле светлого минимализма для частных лиц, которым нужны апостиль, легализация, восстановление и перевод документов в Украине.

---

### Дизайн и стиль

- **Палитра:** белый фон, мягкие кремовые/серые оттенки, тёмно-синий акцент (как на украинских печатях документов), зелёная кнопка WhatsApp
- **Типографика:** крупные элегантные заголовки + чистый sans-serif для основного текста (Inter / Manrope)
- **Атмосфера:** много воздуха, аккуратные карточки, плавные анимации появления при скролле, доверительный официальный вид
- **Адаптивность:** идеально работает на мобильных (большинство клиентов придёт с телефона)

### Форма заявки (сохранение в Supabase / Lovable Cloud)

**Поля формы:**
- Имя
- Телефон (обязательное)
- Email (необязательное)
- Услуга (выпадающий список — все 10 услуг)
- Сообщение / комментарий

**Что происходит при отправке:**
1. Валидация всех полей через zod (длина, формат телефона/email, защита от спама)
2. Заявка сохраняется в таблицу `leads` в Supabase
3. Пользователь видит сообщение «Спасибо! Мы свяжемся с вами в течение 30 минут»
4. Дополнительно показывается кнопка «Написать сразу в WhatsApp» с предзаполненным сообщением

**Где смотреть заявки:**
- Через панель Lovable Cloud → Database → таблица `leads`
- Можно экспортировать в CSV/Excel

**Безопасность (RLS):**
- Анонимные пользователи могут только **создавать** заявки (INSERT)
- Чтение/удаление — только для авторизованных админов (если позже понадобится админка)
- Защита от спама: rate-limit и honeypot-поле
